Acordion

This is the first item's accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.

Eplicacion puntos importantes del acordion

Utilizamos un div el cual hace de funcion accordion, este div tiene la clase accordion y ademas debe tener un id

dentro de este div vamos a tener multiples div mas los cuales representan los items del acordion estos nuevos div deben tener la clase accordion-item

dentro de cada uno de los items vamos a tener la siguiente estructura: un tag H1-5 y luego un tag div que representa el contenido del acordion

El tag H1-5 representara el titulo de la pestania del acordion y ademas sera un botton que nos permita cuando toquemos que el acordion se despliege, para ello necesitamos lo siguiente

  • El tag H1-5 que utilize la clase accordion-header
  • Dentro del tag H1-5 tener un tag button
  •                 El tag button debe tener la siguientes propiedades:
                class="accordion-button collapsed" 
                type="button" 
                data-bs-toggle="collapse"
                data-bs-target="#collapseTwo" 
                aria-expanded="false"
                aria-controls="collapseTwo"
