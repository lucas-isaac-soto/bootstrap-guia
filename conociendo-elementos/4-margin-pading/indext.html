Margins y Paddings

ejemplos

El formato de clases para el Margin y el Padding son el siguiente

  • {property}{sides}-{size} PARA xs
  • {property}{sides}-{breakpoint}-{size} PARA sm, md, lg, xl, and xxl.
Las propiedades pueden ser:

property

descripcion
m

Para clases margin
p

Para clases padding
Los sides pueden ser:

sides

descripcion
t

Para margin-top o padding-top
b

Para margin-bottom o padding-bottom
s

(Start) Para margin-left o padding-left
e

(End) Para margin-right o padding-top
x

Para left y right
y

Para top y bottom
blank

Para los 4 lados
Para el size:

size

descripcion
0

Para eliminar margin o padding
1 Para margin o padding a $space * .25
2 Para margin o padding a $space * .5
3 Para margin o padding a $space
4 Para margin o padding a $space * 1.5
5 Para margin o padding a $space * 3
auto Para margin auto
(Podemos agregar mas espacios agregando mas entradas a $spacers Sass map variable.)
